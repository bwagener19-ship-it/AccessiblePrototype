Test Procedure:
1. Open VS Code.
2. Ensure the Python extension is enabled.
3. Open the Integrated Terminal.
4. Navigate to the project folder:
     cd AccessiblePrototype
5. Run the full test suite:
     python -m unittest discover -v
6. Observe:
     - Passed tests
     - Failed tests
     - Errors
     - Traceback
7. Document results in the Test Progress Report.
